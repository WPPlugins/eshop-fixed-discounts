Extend eShop's site-wide discounts system with the ability to set fixed-rated values.

== Description ==

eShop offers a discounting system that only allows for percentages to be applied to the cart total. This plugin extends it by allowing fixed-rate discounts. 

= For more control... = 

If you'd also like to offer fixed-rate discount codes/coupons, [please consider buying <strong>eShop Coupons Plus</strong>](http://usestrict.net/2013/12/eshop-coupons-plus/).  


== Installation ==

1. Upload eshop-fixed-discounts.zip to your blog's wp-content/plugins directory;
1. Activate the plugin in your Plugin Admin interface;
1. Go to Settings -> eShop -> Discounts to access the modified discount form;
1. Set up the discounts, choosing whether each one is a percentage (%) or fixed-rate ($);
